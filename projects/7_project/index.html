Machine Learning for Nanoporous Graphene Design

Predicting nanopore formation probability and time in graphene using a two-stage ML framework on kinetic Monte Carlo data

Nanoporous graphene is a promising material for gas separation, water desalination, and DNA sequencing — but controlling pore geometry during fabrication remains a key challenge. This project developed a two-stage machine learning framework trained on 20,840 unique graphene nanopore structures generated via kinetic Monte Carlo simulations.

The framework achieves 97% R² for formation probability and 95% R² for formation time, and uses SHAP-based feature importance to quantify the effect of 18 structural descriptors (rim atom count, shape factor, symmetry, etc.) on pore kinetics — providing physically interpretable design guidance.

Published in J. Chem. Phys. (2022, Vol. 156, No. 20) — Editor’s Pick.

(Sheshanarayana & Govind Rajan, 2022)
